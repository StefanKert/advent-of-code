Installation

Install Wasmtime to run WASM files locally:

# Linux/macOS curl https://wasmtime.dev/install.sh -sSf | bash # Windows winget install BytecodeAlliance.Wasmtime
Basic Usage

Run the solver with your input:

# Run Part 1 wasmtime --env AOC_PART=1 \ --env AOC_INPUT="input" Day1.wasm # From file cat input.txt | wasmtime \ --env AOC_PART=1 Day1.wasm
Advanced

Run both parts or measure performance:

# Both parts wasmtime --env AOC_INPUT="$(cat input.txt)" Day1.wasm # With timing time wasmtime --optimize \ --env AOC_PART=1 Day1.wasm < input.txt

Also works with: Wasmer, wazero

Install Wasmtime

  1. Linux/macOS:
    curl https://wasmtime.dev/install.sh -sSf | bash
  2. Windows:
    winget install BytecodeAlliance.Wasmtime
  3. Verify:
    wasmtime --version

Using Environment Variables

# Run Part 1 wasmtime --env AOC_PART=1 --env AOC_INPUT="your input" Day1.wasm # Run Part 2 wasmtime --env AOC_PART=2 --env AOC_INPUT="your input" Day1.wasm

Using File Input

cat input.txt | wasmtime --env AOC_PART=1 Day1.wasm wasmtime --env AOC_PART=1 Day1.wasm < input.txt

Run Both Parts

wasmtime --env AOC_INPUT="$(cat input.txt)" Day1.wasm

Performance

time wasmtime --optimize --env AOC_PART=1 Day1.wasm < input.txt

Other Runtimes

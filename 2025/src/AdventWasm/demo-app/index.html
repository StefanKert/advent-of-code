Run Locally with Wasmtime

Download the WASM file above and run it locally using Wasmtime, a standalone WebAssembly runtime.

Install Wasmtime

  1. Linux/macOS: Run this command in your terminal:
    curl https://wasmtime.dev/install.sh -sSf | bash
  2. Windows: Download the installer from GitHub Releases or use:
    winget install BytecodeAlliance.Wasmtime
  3. Verify installation:
    wasmtime --version

Running the Solver

After downloading the WASM file, run it with your puzzle input:

Option 1: Using Environment Variables

# Run Part 1 wasmtime --env AOC_PART=1 --env AOC_INPUT="your input here" Day1.wasm # Run Part 2 wasmtime --env AOC_PART=2 --env AOC_INPUT="your input here" Day1.wasm

Option 2: Using File Input (stdin)

# Pipe input from a file cat input.txt | wasmtime --env AOC_PART=1 Day1.wasm # Or use input redirection wasmtime --env AOC_PART=1 Day1.wasm < input.txt

Run Both Parts

Omit AOC_PART to run both parts:

wasmtime --env AOC_INPUT="$(cat input.txt)" Day1.wasm

Multiline Input

For multiline input, use file redirection or a heredoc:

# Using heredoc (bash) wasmtime --env AOC_PART=1 --env AOC_INPUT="$(cat <<'EOF' line 1 line 2 line 3 EOF )" Day1.wasm

Performance Profiling

# Use time to measure execution time wasmtime --env AOC_PART=1 Day1.wasm < input.txt # Enable optimizations wasmtime --optimize --env AOC_PART=1 Day1.wasm < input.txt

Other WASM Runtimes

You can also use other WASI-compatible runtimes:

  • Wasmer - wasmer run Day1.wasm
  • wazero - wazero run Day1.wasm
  • aspect - supports WASI natively
